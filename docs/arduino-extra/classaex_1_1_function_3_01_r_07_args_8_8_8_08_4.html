Detailed Description

template<class R, class... Args>
class aex::Function< R(Args...)>

Function wrapper for global functions, lambdas, static methods and instance methods.

Template Parameters
RReturn type of the function
ArgsTypes of the arguments passed to the function
Note
Write the type like Function<void(int, float)> and not Function<void, int, float>.
